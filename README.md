Crowd Funding Platform
A full-stack crowdfunding web application built with Flask, MySQL, HTML, CSS, JavaScript, and Jinja templates. Users can create projects, support projects through donations/backings, manage profiles, and explore campaigns through a working search system.
Features


User Authentication (Register/Login/Logout)


Role-based users (Creators & Backers)


Create and manage crowdfunding projects


Backing/Donation system


Dynamic dashboards


Project search functionality


Image upload and rendering


AJAX/JavaScript interactions


MySQL database integration


Responsive frontend design

Installation


Clone the repository


git clone <your-repository-link>cd crowd-funding


Create a virtual environment


python -m venv venv


Activate the virtual environment


Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate


Install dependencies


pip install -r requirements.txt


Configure the database


Create a MySQL database and import the SQL schema.
Update your database credentials in the Flask configuration file.


Run the application


python app.py

This project was built mainly for learning and showcasing backend development skills using Flask and MySQL while also improving frontend integration and database design knowledge.
